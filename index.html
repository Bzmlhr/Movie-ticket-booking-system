Movie Ticket Booking System

Welcome to the Movie Ticket Booking System

This project aims to develop and optimize an efficient movie ticket booking system, with improved user registration and login options using phone number verification . The system will ensure a seamless user experience, incorporating modern authentication protocols like OAuth 2.0 for security.

Project Highlights

  • Phone number registration with verification code login
  • OAuth 2.0 authentication for secure login
  • Optimized and responsive UI for a better user experience
